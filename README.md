## How Works

1- Open the laravel project folder as a root folder otherwise you will need to edit the "User Setting" in the section below ðŸ‘‡(https://github.com/mahmoudshahin1111/laravel-routes/blob/master/README.md#user-settings) 
2- Press LEFT CTRL + SHIFT + P
3- Choose `LaravelRoutes: Start` 
4- Press Enter.
5- Try to write any route path inside double quotes `"/* Your route Here */"` you will see a list of suggested routes with the same name 


## User Settings
```json
    "routes.RoutesFolderPath": "/routes",
    "routes.ProvidersFolderPath": "/app/Providers"
```
